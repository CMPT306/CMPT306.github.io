Question 1 (Easy Level)
Solve change-making problem which we have discussed in class. Please read the 12th slide for the description of the question. Given the amount n=10 and coin denominations = [1,3,4,5,6]. Find the minimum of coins whose values add up to n.

Question 2 (Meidum Level)
There are 30 steps in a stair. You can either climb 1 or 2 steps each time. Find how many different ways can you climb to the top?

Question 3 (Difficult Level)
You roll a fair six-sided dice N times. The sum of the values is M. How many different ways can you roll the dice? Find answers for there three cases:
1) N=2, M=7
2) N=3, M=9
3) N=8, M=24
