Lab 5: Brute Force - Convex Hull

This lab will involve implementing a brute force solution to the convex hull problem.

Begin with the following classes:

  • ConvexHull.java - a collection of points, where a convex hull will be calculated. This is the class you need to modify.

  • Edge.java - the edge between two points. This class calculates a line equation, distance, and the result of plugging in any point into the line's general line equation.

  • ConvexHullGUI.java - a GUI to make it easier to debug your program. This was adapated and modified from another GUI for this lab, so it's not the best example of coding. It invokes the necessary methods in the ConvexHull class.

Step 1. Download the three classes.

Examine the code, especially the ConvexHull and Edge classes. You will see a private calculateConvexHull() method at the end of the ConvexHull class that you need to complete. Also make sure you familiarize yourself with the existing methods in ConvexHull and Edge.

It is important to notice the current calculateConvexHull() method does not work at all! It's only slightly better than a stub method. Try it out, and you'll see!

Step 2. Agree on pseudocode.

With your partner, review the convex hull problem and agree on pseudocode to calculate a convex hull using the brute force technique.

This is an important step, because the algorithm has so many steps.

Step 3. Code  your solution.

Implement your algorithm. The ConvexHullGUI program will help you decide if your solution works.

Finish Early? Implement the Closest Points algorithm

The current closest points implementation in the ConvexHull class (the getClosestPoints() method) returns the first two points in the convex hull. Modify it so it correctly returns an array of the two closest points in the collection of points.

Your changes should automatically appear in the ConvexHullGUI program.
