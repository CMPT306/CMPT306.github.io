This lab will involve writing a solution to the classic game of Hangman where players get 10 guesses to guess a word chosen by random from your game.

The following file contains a Python script that is passed  file containing the words to be chosen from (words.txt)

- (Jupyter Notebook User) Hangman.ipynb.zip

- (Eclipse User) Hangman.py.zip

Please use at least one map() and one lambda operator. You can use this page as reference. I will not give you full points if you do not use any map or lambda.

The game can be written in any style that you choose. The important thing is that you account for the following:

- only 10 times guesses are allowed.

- only allow alphabetic characters to be considered. If a user enters a '$', '*', etc. skip over the input. (Python has a pretty useful function to check if a string is alphabetic.). In this case, print "Only allow alphabetic characters."

- the file you are choosing words from (words.txt) contains all lower-case characters. You can allow a user to enter an upper-case alphabetic character (A .. Z), but get its lower-case equivalent when comparing.

- if a user enters a character that appears more than once in the word (i.e. 'aardvark' where 'a' occurs multiple times) be sure to identify each occurrence of the character, and not just the first.

- allow a user to guess a character only once. If they enter the same character more than once, print out a message indicating the character has already been considered. Furthermore, if the character is an incorrect guess, only count it as an incorrect guess the first time. In this case, print "The letter", ch, "has already been used."

- if a user enters a character that is not in the correct word, print ch,"does not occur."

- if scuess, print 'Congratulations!', otherwise, print 'Sorry dude, the word is', word
