Lab 7: Divide and Conquer

This lab will involve implementing a divide and conquer solution to the convex hull problem. Begin with the following classes:

  • PointSet.java - a collection of points. This is the class you need to modify.

  • Line.java - the line between two points. This class is similar to the Edge class from the previous lab, but it now has additional methods to calculate the area of the triangle between three points and whether a third point is above (i.e. left) or below (i.e. right) the line.

  • QuickHullGUI.java - a GUI to make it easier to debug your program. It calls the necessary methods in the PointSet class.

 

Step 1. Review the algorithm.


Select a partner on your own and sign up.

Step 2. Download the three classes.

Create an Eclipse project, and download the above three classes and place them into your project. 

Examine the code, especially the PointSet and Line classes. You will see two private methods in the PointSet class that you need to complete: calculateConvexHull() and calculateUpperHull(). Also make sure you familiarize yourself with the existing methods in PointSet and Line.

Several methods that you will find helpful, including:

Line:
  • isBelow(Point other): returns true if the line is below a point, false otherwise
  • isAbove(Point other): returns true if the line is above a point, false otherwise
  • compareTo(Point other): return +1 if the line is below a point, -1 if the line is above a point, and 0 if a point is on the line
  • getDeterminant(Point other): returns the negative determinant, as shown on page 197
  • getTriangleArea(Point other): returns the area of the triangle created from the endpoints of the line and the other point
  • reverseLine(): returns a line where the endpoints are reversed.
PointSet:
  • calculateQuickHull(): this method determines the leftmost and rightmost points, creates an ArrayList of all the other points in the set, and calls your calculateConvexHull() method (to be written)
  • calculateLowerHull(Line line, ArrayList pointsBelow): calculates the lower half of the convex hull by using the stub method calculateUpperHull()
  • getPointsAbove(Line line, ArrayList set): returns a list of all the Points that are above the line
  • findMaxPoint(Line line, ArrayList set): returns the Point farthest from the line
  • findLeftRight(): returns the leftmost and rightmost Points by using the divide and conquer method described and analyzed in class.

Note: the textbook uses the language "if a point p3 is to the left of line p1p2". However, when you call methods, the relationship is reversed, as in isBelow(p3) in the Line class. These actually mean the same thing: when a point p3 is to the left of the line, then that point is above the line, and of course the line is below the point.

Second Note: In the calculation of the determinant shown on page 197, it indicates that a positive sign of the determinant indicates that a point is to the left (or above) the line. However, Java orders points on the Y axis in a downward fashion (i.e. the values of the Y axis decrease as you move up the Y axis.) To compensate for this, I have reversed the sign of the determinant. This shouldn't really affect you - it turns out that if the determinant is reversed, the convex hull is drawn clockwise instead of counter-clockwise. (There is a main() method in the Line class that illustrates the value of the determinant given 3 points.)
