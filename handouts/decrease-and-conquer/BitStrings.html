Bit Strings

For each element ai, we associate biti with it. (We read bit positions from left to right.)

If biti = 0, ai is not in the set. If biti = 1, ai is in the set.

To generate all subsets of an n-element set, we generate successive binary numbers from 0 ... (2^n)-1.
