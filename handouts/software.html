Python is a distinctly different programming language than Java. Although Python can be written using object-oriented techniques such as classes, the main distinction is that a given programming problem can be solved in significantly fewer lines of Python code. Python is not typed and allows variables to be declared dynamically. There are no begin-end squiggly braces, and indentation is used to identify blocks of code. By default, Python is loaded with several different data structures including lists, tuples, sets, and dictionaries. A good rule of thumb when developing in Python is to look for simple solutions to common programming problems; quite often we make things more difficult than necessary and Python allows a simple and elegant solution that may not be obvious at first to the beginning Python developer.

A great place to begin working with Python is the Python tutorial.
