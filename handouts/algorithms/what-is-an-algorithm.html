What is An Algorithm?

No universal definition, but here goes:

An algorithm is a sequence of unambiguous instructions for solving a problem.

Consider the problem finding the largest element in a list of size N:

An algorithm may be written in:

Natural Language:

1. Select the first element as the current largest.

2. Compare each successive element to the current largest. If an element is larger than the current largest, set the current largest to that element.

3. Report current largest as the largest element

Pseudocode:

1. largest = list[0]

2. i = 1

3. while there are more elements in list:

    3a. if list[i] > largest then largest = list[i]

    3b. i = i + 1

4. print largest

Important Points

1. Each step must be unambiguous

2. The range of inputs must be carefully determined

3. The same algorithm can be expressed several different ways

4. There may exist several algorithms for solving the same problem
